💼 项目经验

DreamScope

▼

DreamScope 是一款基于 Next.js 全栈架构开发的移动端梦境记录与分析应用。通过集成 DeepSeek AI 智能分析技术，帮助用户记录、分析和理解自己的梦境世界，提供个性化的心理健康调察和梦境解读服务。

技能分析：

• 使用React18 + Next.js进行全栈开发，支持服务端渲染和静态生成，优化首屏加载性能
• 使用lib-flexible + postcss-pxtorem实现移动端适配，实现多设备兼容
• 集成 DeepSeek API 实现 AI 对话的流式输出；对接 Coze 工作流 API 完成 AI 头像生成功能
• 设计基于 Zustand 的全局状态管理方案，通过自定义 Hooks 封装核心业务逻辑，实现 Store 与 UI 组件的低耦合
• 采用瀑布流布局，提高页面内容展示密度和阅读流畅性

项目地址：https://github.com/LgFvm9353/DreamScope

TransVoice

▼

基于浏览器端本地化运行的文本转语音（TTS）应用 Demo，使用 Hugging Face Transformers.js 库与 Xenova 的 SpeechTTS 模型，无需服务器支持。

技能分析：

• 采用单例模式管理 TTS 模型加载，有效避免资源重复加载消耗
• 利用 Web Worker 封装模型推理计算，将密集型任务移出主线程，保障页面交互流畅性
• 用 Tailwind CSS 快速构建响应式 UI，提升前端开发效率
• 通过 Promise.all 并行加载主模型及其 Embedding 文件，显著缩短应用初始化时间，优化用户体验

项目地址：https://github.com/LgFvm9353/TransVoice

云帐通

▼

基于React 18和Zarm UI开发的移动端智能记账应用，具备账单管理、收支分析、数据可视化等功能，并集成AI智能对话服务，提供个性化的财务建议和专业的收支分析。

技能分析：

• 封装 Axios 请求库并实现基于 JWT 的双 Token 认证机制（Access/Refresh Token），实现用户认证的无感刷新
• 接入专属Agent并连接Coze数据库，提供基于真实交易数据的智能分析与专属对话
• 使用 React.lazy + Suspense 实现懒加载，自动拆分代码并按需加载组件和路由，显著提升页面加载速度
• 开发自定义 Hook 管理流式 AI 对话，支持实时消息渲染、聊天记录本地持久化及自动滚动
• 集成 ECharts 实现数据可视化，支持饼图、柱状图等多种图表展示形式

项目地址：https://github.com/LgFvm9353/CloudBillPass
